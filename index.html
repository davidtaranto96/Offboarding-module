ℹ️ What is this tool? This tool allows JBKnowledge IT admins to securely offboard Microsoft 365 users: back up their OneDrive and emails to Azure Blob Storage, remove licenses and group memberships, and block sign-in — all in one auditable workflow. Requires Global Admin or equivalent permissions.
📖
How to use this tool ▾

Follow these steps in order to safely offboard a Microsoft 365 user:

1
Search the user Type the employee's name or email below and click Search. Verify the account before proceeding.
2
Review the checklist The pre-offboarding checklist will auto-run and show OneDrive size, mailbox count, group memberships, and license status.
3
Run the backup Click Start Backup to copy OneDrive files and emails to Azure Blob Storage. Already-uploaded files are skipped automatically.
4
Execute offboarding Select which actions to apply (block sign-in, remove licenses, remove groups), add notes, then click Execute Offboarding. A confirmation email will be sent to IT.

⚠️ This action is irreversible once sign-in is blocked. Always complete the backup before executing offboarding.
